# AI Resume Builder with ATS Scoring

An AI-powered resume builder that analyzes resumes against job descriptions and provides an ATS (Applicant Tracking System) score along with keyword matching insights. Built using Streamlit for an interactive user interface.

## Overview

This project helps users improve their resumes by comparing them with job descriptions and identifying missing keywords. It calculates an ATS score and provides suggestions to make the resume more job-ready.

## Features

- Upload resume (PDF format)
- Input job description
- ATS score calculation
- Keyword matching analysis
- Suggestions for improvement
- Clean and interactive UI using Streamlit

## Technologies Used

- Python
- Streamlit
- Natural Language Processing (NLP)
- Scikit-learn / Text Processing
- PDF processing libraries

## How It Works

1. User uploads a resume
2. User inputs a job description
3. System extracts text from both inputs
4. Performs keyword matching and similarity analysis
5. Calculates ATS score
6. Displays score and improvement suggestions
